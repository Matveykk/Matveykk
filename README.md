## 📌 Проекты

### ☕ Java

**[Expense Splitter API](https://github.com/Matveykk/expense-splitter-api)** — REST API для деления групповых расходов (по типу Splitwise) с алгоритмом упрощения взаимных долгов (min-cash-flow), сводящим расчёты к минимуму переводов.
`Spring Boot` `Spring Security + JWT` `PostgreSQL` `JPA` `Swagger` `Docker Compose` `JUnit`

**[Concurrent Cache](https://github.com/Matveykk/concurrent-cache)** — потокобезопасная кеш-библиотека на чистой Java: вытеснение по LRU и истечение по TTL, метрики, builder. Без внешних зависимостей.
`Java 21` `Concurrency` `Data Structures` `JUnit`

**[Expression Evaluator](https://github.com/Matveykk/expression-evaluator)** — парсер и вычислитель математических выражений: рекурсивный спуск, корректный приоритет операций, переменные и функции.
`Java 21` `Recursive-descent parser` `AST`

**[Link Checker](https://github.com/Matveykk/link-checker)** — многопоточный проверяльщик битых ссылок: обход сайта пулом потоков на `HttpClient`, отчёт в CSV, коды возврата для CI.
`Java 21` `ExecutorService` `java.net.http`

**[Kanban API](https://github.com/Matveykk/kanban-api)** — REST API канбан-доски с JWT-аутентификацией. `Django REST Framework` `PostgreSQL` `Docker`

**[Vacancy Dashboard](https://github.com/Matveykk/vacancy-dashboard)** — сбор и анализ вакансий с hh.ru с визуализацией. `pandas` `matplotlib` `requests`
